# Aethera Admin

This is the **Admin Application** companion to the main Aethera e-commerce user app. It is designed to manage and oversee the operations of the user-facing platform. Built with Kotlin, Jetpack Compose, and modern Android development practices.


## 🏗️ Architecture
This project follows the **MVVM (Model-View-ViewModel)** architectural pattern to separate concerns and make the codebase scalable and maintainable.

## 🛠️ Tech Stack & Libraries
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Dependency Injection**: Hilt
- **Navigation**: Jetpack Navigation Compose
- **Network**: Retrofit & Gson
- **Image Loading**: Coil
- **Serialization**: Kotlinx Serialization JSON
- **Backend & Database**: Firebase (Firestore, Storage)

## 📦 Features (Current Stage)
*Initial phase of development.* The application is set up with core dependencies and modular structure to handle typical admin app capabilities for an e-commerce platform.

## 🚀 Getting Started

1. Clone the repository.
2. Provide your own `google-services.json` file from Firebase Console and place it in the `app/` directory.
3. Build and run the project using Android Studio.

## ⚠️ Notes
- `google-services.json` and local error logs are ignored intentionally from the version control for security and cleanliness.
